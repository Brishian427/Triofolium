Project Trifolium is a self-improving algorithmic trading system built on a multi-LLM architecture (NVIDIA Nemotron + Anthropic Sonnet). A Navigator-Architect-Coder pipeline explores strategy design space through an Element Periodic Table decomposition, evaluated by a 9-section framework derived from IMC competition experience. Live deployment uses a profit harvester with institutional Risk Gate governance (12,619 logged decisions, 7 hard-kill layers). During operation, the system discovered two constitutional design refinements and produced a natural experiment comparing algorithmic vs discretionary execution, demonstrating that principal-risk is system-risk, and that institutional governance must extend to the human operator.
