Constitutional refinements + next iteration

Refinements discovered live

  • Rate limits must distinguish risk-increasing from risk-reducing actions.
  • Session gates must not encode the operator's real-world calendar.
  • Manual positions require explicit ownership and automation opt-in.

Next iteration

  • Principal-protection layer: exposure caps, cool-downs, override locks.
  • Absolute-return target formulation instead of only cross-sectional rank.
  • Momentum context features and multi-horizon prediction.
